# AI Assistant Skills

This directory contains "skills" for AI assistants. Each skill is a markdown file that provides detailed, step-by-step instructions for an AI to perform a complex task related to the Oakestra project.

For example, `troubleshoot-oakestra.md` contains a detailed guide for an AI to diagnose and troubleshoot a live Oakestra deployment.

These skills are designed to be used by AI agents to automate and assist with development and operations tasks.
